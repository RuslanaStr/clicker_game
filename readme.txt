  localStorage

Механика:

  1. Дополнительная жизнь за бонусы.
  При накоплении 400 бонусов - можно пополнить уровень жизней на 1 секцию, после пополнения - 
  бонусы списываются.
  2. Пробел 'Space' - для включения брони.

Клавиши:

1. Пробел 'Space' - для включения брони.
2. Цифра 4 'Digit4' - для пополнения жизни при условии накопления 400 бонусов.
3. Для выбора оружия нужно нажать соответствующие клавиши:
         - цифра 1 'Digit1' (первый вид оружия);
         - цифра 2 'Digit2' (второй вид оружия);
         - цифра 3 'Digit3' (третий вид оружия);
